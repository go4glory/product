== Description ==

Create website baby name listing easy with "Baby Name Finder" for WordPrerss,  

### Baby Name Finder by http://pickplugins.com
* [Live Demo!&raquo;](http://www.pickplugins.com/demo/baby-name-finder/)

Use following shortcode to display baby name list on any page content
`[baby_name_list]`

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Baby Name Finder</strong>" activate it.<br />
